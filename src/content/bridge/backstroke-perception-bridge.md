## 前提：仰式的感知本質

仰式有一個其他三式都沒有的感知挑戰：**方向感的缺失**。臉朝上、看不到前進方向、耳朵入水後聽覺改變——這些對身體來說都是陌生的訊號，即使動作完全正確，感知系統一開始也難以接受這個狀態是安全的。

所有仰式感知建立的起點，是先讓身體接受「仰躺在水裡是安全的」。這一步沒有完成之前，後續所有感知教學效率都很低。

仰式另一個核心命題：**旋轉是速度的乘數，且旋轉幅度在加速時不縮小**（這與自由式正好相反）。「加速時縮小旋轉以提高划頻」的策略在仰式不適用——旋轉幅度的穩定性本身就是仰式效率的特徵。

仰式的 C 型不是「全身張力遮蔽感知」（那是自由式 C 型），而是**感知不到身體偏離水平位置的訊號**——頭太高、腰弓起、腳踢不到水面，三個任何一個跑掉，阻力都會明顯增加，但學生常常感覺不到。

## 感知時刻一：方向感與仰漂安全感（前置/L1）

**物理現實**
仰漂時浮力是真實的，但感知系統需要時間學會信任它。耳朵入水後聽覺改變、視線朝上失去空間參照——這些訊號對神經系統來說代表「異常」，即使身體安全，緊張反應仍然會自動出現。

感知目標不是「放鬆」（要求放鬆通常讓人更緊張），而是讓學生在仰躺的過程中**感覺到水在撐著自己**，而不是自己在撐自己。

**教練觸發條件**
浮板仰漂：讓下腹去找浮板。浮板是移除「擔心沉下去」的背景焦慮，不是支撐工具。三層感知掃描：胸口（浮力最明顯）→ 耳朵（聲音變化）→ 腳（水面托感）。

**學生說這個 → 有效訊號**
- 「感覺水在撐著我」
- 「耳朵裡的聲音變了」
- 「胸口吸氣時好像往上一點」
- 「還好，沒有想像中可怕」（安全感建立的間接訊號）

**學生說這個 → 聽起來像，但要再問**
- 「我有放鬆」→ 問「放鬆的時候，有沒有感覺到什麼在撐你？」放鬆是手段，感覺到浮力才是目標。
- 「感覺快沉了」→ 通常是頭部位置問題（頭太高 → 髖下沉），不是浮力不夠。問「胸口有沒有感覺到浮力在起伏？」把注意力引向浮力最明顯的地方。
- 「感覺OK」→ 太籠統，繼續問三層掃描的任何一層，確認感知有真正打開。

## 感知時刻二：仰漂三指標（C 型核心）

**物理現實**
仰式的身體位置感知，比其他三式都更難自我監控，原因是臉朝上、失去視覺參照、耳朵入水後本體感知準確度下降。

三個指標同時成立，代表身體在最低阻力位置：
- **耳朵在水裡**——頭部高度正確，不過高也不過低
- **額頭不碰水**——頭部不過低（過低 = 下巴收過頭，水蓋上來）
- **腳踢得到水面**——腰部沒有弓起，下半身水平

三個指標任何一個跑掉，都有物理後果。但學生最難感知的是**腰弓起**——腰弓起時感覺像在用力游，但實際上阻力在增加。

**教練觸發條件**
靜止仰漂先確認三指標，再加入踢腿確認，再進入完整仰式。每個階段都先確認指標還在，再進入下一步。

**學生說這個 → 有效訊號**
- 「耳朵在水裡但額頭沒有水」
- 「腳感覺踢得到水面」
- 「加速後感覺阻力變大，應該是哪個指標跑掉了」（能察覺 = C 型感知建立中）

**學生說這個 → 聽起來像，但要再問**
- 「感覺位置對」→ 最不可靠的描述。問三個指標：「耳朵在水裡嗎？額頭有沒有感覺到空氣？腳踢得到水面嗎？」三個都能答才算。
- 「慢游還好，一加速阻力就很大」→ C 型最典型描述。問「加速之後，這三個指標哪一個先跑掉？」幫助學生建立更精確的觀察。
- 「感覺身體一直在下沉」→ 通常是頭部過高（頭高 → 髖下沉）。不是沉的問題，是頭的問題。

## 感知時刻三：上踢推進感知

**物理現實**
仰式踢腿的推進主力是**上踢**（足背朝後朝上），對應自由式的下踢。下踢是回收動作，另一隻腳的重量自然帶它下去，不需要刻意控制。

感知目標：上踢時感覺「一團水被往上推，身體往前走」。踢腿的推進不是「我的腿往後踢」，而是「水被移走了，身體繼續前進」。

踝關節蹠屈幅度是硬體邊界——踝關節僵硬時足背無法朝後，嚴重時上踢反而產生阻力而非推力。

**教練觸發條件**
手持浮板仰式踢水（手部固定，感知資源全在腿部），踢完後感覺身體有沒有繼續往前。或單腳練習，一次專注一隻腳的水團感知。

**學生說這個 → 有效訊號**
- 「上踢的時候感覺一團水被往上推」
- 「踢完之後身體還在往前走」
- 「感覺腳踝要往上甩，不是膝蓋在蹬」

**學生說這個 → 聽起來像，但要再問**
- 「我有用力踢」→ 在描述肌力輸出。問「踢完之後，身體有沒有繼續往前走？」
- 「感覺不到什麼」→ 先確認踝關節柔韌性。讓學生坐姿做踝關節蹠屈，確認腳掌能否充分朝向斜後方。如果不能，是硬體問題。
- 「踢了但身體沒有往前」→ 兩種可能：踝關節不夠柔（硬體），或膝蓋彎曲過大（感知目標錯誤——「腳踝帶動」不是「膝蓋帶動」）。

**解剖邊界注意**
踝關節柔韌性訓練優先於腿部力量訓練。踝關節極度僵硬的泳者，踢腿在無手臂輔助時甚至可能產生向後的淨力（倒退）。蛙鞋訓練和陸上伸展是最直接的方法。

## 感知時刻四：入水角度感知

**物理現實**
仰式入水：小指側先行，沿肩線延伸方向，角度 30-45°，不跨越中線。

小指先入水有三個邏輯：肩旋轉的自然延伸、入水阻力最小、捕水位置預設（入水後手掌自然轉向推進方向）。

跨越中線入水是最常見錯誤：誘發蛇行偏移，橫向動量分量增加，旋轉軸偏移，能量散逸。

**教練觸發條件**
以「肩線延伸」為基準，感知「手臂沿身體側面前方放入水」——不是從上方落下，是從側面延伸進去。

**學生說這個 → 有效訊號**
- 「感覺手沿著肩線延伸進去」
- 「小指先入水感覺比較滑」
- 「入水的時候沒有感覺到側向的力」

**學生說這個 → 聽起來像，但要再問**
- 「我有把手放直進去」→ 問「入水之後，游的路線有沒有歪？」跨越中線的人通常感覺不到自己在蛇行。
- 「感覺手臂是直的」→ 描述手臂形狀，不是入水位置。問「入水的時候，你能感覺到手是在肩膀前面進去，還是有越過鼻樑？」

## 感知時刻五：捕水感知（EVF）

**物理現實**
仰式捕水和自由式邏輯相同：高肘讓前臂面向推進方向，手掌是主推進面，不是前臂。

仰式特殊之處：捕水在身後（不是身前），手掌在肩線後方約 10-15cm 建立高肘，前臂面向推進方向。正確感知是「拉身體過去」，不是「推水出去」。

「手臂直線推水」是最常見的仰式划手錯誤——有效槳面積縮小，感覺在用力但推進力下降，且肩部剪力增加。

**教練觸發條件**
Sculling 建立前臂水壓感知，再做捕水動作：感覺前臂整個面對水產生的水壓，而不只是手掌心。

**學生說這個 → 有效訊號**
- 「划手感覺像在拉繩子，不是推水」
- 「前臂有一個面的感覺，不只是手掌」
- 「手掌方向感覺是朝著腳的方向」（推進方向正確）
- 「拳頭張開那一刻，手掌突然有東西」（復活感）

**學生說這個 → 聽起來像，但要再問**
- 「我有用力划」→ 在描述肌力。問「划的時候，感覺是在推水走，還是身體在被拉過去？」
- 「感覺手臂在往後甩」→ A 型 L5 的典型代償——旋轉縮小後手臂改用「甩」的方式代償，維持動作感但沒有真正抓到水。問「甩過去的時候，有沒有感覺到什麼阻力？」

## 感知時刻六：旋轉不對稱感知（最容易搞錯方向的感知）

**物理現實**
仰式的旋轉不是兩側對稱地轉 45°，而是有方向性的不對稱：

- **回臂側肩膀**：主動用力，盡量向上抬高，接近觸及下巴
- **划水側肩膀**：被動深沉，是回臂側上抬的自然結果

這是最容易搞錯方向的感知：「把划水側肩膀往下壓」和「把回臂側肩膀往上抬」，從外部看起來幾乎相同，但感知方向完全相反。主動用力往下壓，會製造肩部張力，長期可能增加傷害風險；主動往上抬，動作更自然，深沉是副產品。

**教練觸發條件**
站立模擬回臂動作：「回臂的手臂，盡量讓肩膀往上抬，靠近下巴」。感覺另一側肩膀有沒有自然地跟著往下。

**學生說這個 → 有效訊號**
- 「抬回臂側肩膀，划水側自然就沉下去了」
- 「感覺是一側往上，另一側被帶著走」
- 「肩膀抬高的感覺，比壓下去輕鬆」

**學生說這個 → 聽起來像，但要再問**
- 「我有把肩膀壓下去推水」→ 方向錯了，但結果看起來一樣。問「壓下去的時候，感覺肩膀有沒有比較緊？」換成「抬高另一側」試試看，比較哪個感覺更自然。
- 「旋轉感覺是兩側對稱在轉」→ 可能是有效的也可能不是。問「主動用力的感覺，是在哪一側？」

## 感知時刻七：旋轉幅度穩定感知（加速不縮小）

**物理現實**
仰式在速度增加時，旋轉幅度維持不縮小（Gonjo 2021）——這是仰式和自由式最關鍵的差異。自由式高速時肩旋轉有縮小趨勢，但仰式的旋轉幅度穩定性是功能需求，疲勞不應作為縮小旋轉的理由。

疲勞時的仰式代償模式：維持肩旋轉，但增加髖旋轉（核心穩定性下降，旋轉從肩轉移到髖）。結果是看起來還在旋轉，但推進效率下降了。

**學生說這個 → 有效訊號（L4-L5）**
- 「加速的時候旋轉感覺和慢游一樣大」
- 「速度快了，旋轉還在，不用刻意維持」

**學生說這個 → 聽起來像，但要再問**
- 「加速的時候旋轉變小了才能划快」→ 這個直覺在仰式是錯的。問「划頻提高之後，划手有沒有感覺推進力變弱了？」旋轉縮小通常伴隨推進力下降。
- 「游快的時候感覺腰在轉」→ 可能是髖旋轉代償——肩旋轉縮小後髖接管。問「感覺肩膀還在旋轉，還是主要是腰在轉？」

## 感知時刻八：方向感知（身體繞頭旋轉，不是頭跟著旋轉）

**物理現實**
仰式方向控制的核心感知：**身體繞著頭旋轉，頭是旋轉軸，不動**。頭如果跟著身體旋轉，方向感立刻消失，路線開始蛇行。

這個感知和大多數人的直覺相反——多數初學者覺得「身體在轉，頭當然也要跟著」。但頭固定才是仰式方向穩定的前提。

方向感在 L5 是一個完整的感知層次：從「慢游方向還好，速度快了就歪」進化到「游 25m 中途幾乎不需要修正方向」。

**教練觸發條件**
讓學生盯住天花板上一個固定點，確認那個點在整趟游泳中是否移動。頭動了，那個點就會移動。

**學生說這個 → 有效訊號**
- 「天花板那個點一直在，沒有跑掉」
- 「感覺身體在轉，頭沒有跟著動」
- 「這趟游完，沒有特別修正路線」

**學生說這個 → 聽起來像，但要再問**
- 「我有盯著天花板」→ 問「游完之後，停下來的位置有沒有偏？」盯著天花板不代表頭完全不動。
- 「感覺頭有轉一點」→ 這個是誠實的描述。問「轉的時候，游的路線有沒有開始歪？」讓學生建立「頭動 → 路線歪」的感知連結。

## 感知時刻九：流線型感知（出發/轉身）

**物理現實**
仰式在四式中，出發和轉身佔整場時間的比例最高——100m 仰式中出發加轉身合計約佔整場 1/3（González-Ravé 2025）。流線型品質是 100m 仰式成績的最重要單一技術預測因子之一。

感知挑戰：出發和轉身後身體速度最高，阻力增長最快（F ∝ v²）。此時流線型的效益最大，但感知也最困難——速度太快，細微的姿勢偏差很難感知到，且頭部已完全入水。

最常見錯誤：流線型滑行期間提前浮頭——感覺像在等待，主觀上很難抑制，但速度損失是實質的。

感知目標：感覺速度開始下降到接近游進速度時，才開始第一划——不是計時，而是感覺速度窗口。

**學生說這個 → 有效訊號**
- 「感覺滑行到速度慢下來才開始划」
- 「頭在水裡，感覺身體還在動」
- 「流線型期間感覺阻力很小，像在滑」

**學生說這個 → 聽起來像，但要再問**
- 「我等了一下才划」→ 問「等的時候，你有感覺速度在變嗎？還是只是在計時？」把感知從時間引向速度感。
- 「感覺在水裡快沒氣了」→ L1-L2 的安全感問題。提前浮頭通常不是意識層面的選擇，而是身體的反射。先在練習中縮短流線型滑行距離，慢慢延長，讓感知系統學會信任。

## 疲勞崩潰的感知順序

仰式疲勞崩潰的代償模式與其他三式不同，有文獻確認的特殊結構：

1. **髖旋轉增加（代償，不是崩潰）**——核心穩定性下降，旋轉從肩轉移至髖。看起來還在旋轉，但推進效率下降。感知：「感覺腰在帶動旋轉，肩膀反而沒什麼感覺」
2. **捕水深度下降**——旋轉質量下降，手臂無法達到最深延伸位置。感知：「感覺划手越來越短」
3. **踢腿頻率降低**——從六拍退化至四拍或兩拍
4. **手臂直線划水**——完全依賴上肢力量，旋轉連動放棄

**關鍵差異：仰式的第一崩潰是代償，不是放棄。** 學生還在「努力游」，外觀上看起來還在旋轉，但旋轉的質量和來源已經改變了。這是仰式最難從外部診斷的崩潰層。

## 三型快速識別

| 學生說 | 通常是哪型 | 第一個問題 |
|--------|-----------|-----------|
| 「划手感覺像在滑水，推不到東西」 | A 型 | 「拳頭游和張開手游，速度差多少？」 |
| 「踢了很多但前進很慢」 | B 型 | 「上踢的時候，有沒有感覺一團水被推走？」 |
| 「慢游還好，一加速阻力就變大」 | C 型 | 「加速後——耳朵還在水裡？額頭不碰水？腳踢得到水面？哪個先跑掉？」 |
| 「感覺手在甩出去」 | A 型 L5 | 「甩過去的時候，有沒有感覺到前臂有阻力？」 |
| 「游完路線歪掉了」 | C 型 或 B 型 | 「歪的時候，頭有沒有跟著轉？」（C型）/ 「踢腿有沒有對稱？」（B型）|
| 「加速時感覺腰在帶旋轉」 | 疲勞代償 | 「肩膀還在旋轉嗎？還是主要是腰在轉？」 |

## 指導語有效性速查

| 常見指導語 | 問題 | 有效替代 |
|-----------|------|---------|
| 「保持平躺」 | 生物力學上對應最低效率的游泳狀態；沒有旋轉就沒有推進 | 「讓回臂側肩膀盡量往上抬」 |
| 「把划水側肩膀往下壓」 | 方向正確但感知錯誤——製造肩部張力；應從回臂側主動 | 「抬高另一側肩膀，這側自然就沉了」 |
| 「頭跟著身體轉」 | 頭轉動 → 方向感立刻消失 → 路線蛇行 | 「盯住天花板一個點，身體繞著頭轉」 |
