# 第一章：背景與發展架構

## 1.1 適當運動員發展（AAD）

Swimming Canada 於 **2008 年**首次發布《長期運動員發展策略（LTADS）》，確立各發展階段的目標與活動。最初的四大策略方向為：

- **競賽審查（Competition Review）**
- **各階段技能檢核表（Stage Skills Checklists）**
- **對俱樂部與家長的指引（Implications for Clubs and Parents）**
- **NCCP 教練教育整合（NCCP Integration）**

過去十年間，這些策略陸續落實：

- **2016 年**：推出「競賽改善計畫——全國賽事」，新增**高峰表現窗口（Peak Performance Windows）**、調整全國賽事時程、規範競賽年齡。
- **2018 年春**：完成第二階段競賽審查，涵蓋省級與發展級賽事建議，以 ADM 技術／戰術／策略支柱為基礎。

**ADM 的服務對象**：教練、游泳選手、家長、俱樂部、整合支援團隊（IST）、裁判、省級體育組織、地方社群。

---

## 1.2 長期運動發展模型（LTD）

**LTD 模型**（前稱 LTAD）是「為每位兒童、青少年與成人提供最佳運動與身體活動參與機會的發展框架，涵蓋成長、成熟、發展能力與運動系統整合。」

### 發展階段

```
Active Start（主動起步）
    ↓
FUNdamentals（基礎樂趣）
    ↓
Learn to Train（學習訓練）
    ↓
Train to Train（訓練為訓練）    ← Podium Pathway 起點
    ↓
Train to Compete（訓練為競賽）
    ↓
Train to Win（訓練為勝利）
    ↓
Active for Life / Competitive for Life / Fit for Life（終身活躍）
```

| 階段 | 說明 |
|------|------|
| **Active Start** | 游泳是唯一不直接負責此階段的項目；基礎游泳由紅十字會負責 |
| **FUNdamentals** | 介紹基本動作技能（agility、balance、coordination），強調樂趣 |
| **Learn to Train（L2T）** | 漸進式挑戰訓練，為專項訓練打基礎 |
| **Train to Train / Compete / Win** | Swimming Canada 全國計畫核心，合稱**獎牌台路徑** |

---

## 1.3 獎牌台路徑（Podium Pathway）

> 「獎牌台路徑旨在培養能在最高層級國際競賽中獲勝的高水準運動員，並在奧運、帕運或世界錦標賽中為國家奪牌。」

| 發展階段 | 目標成就 | 對應賽事 |
|---------|---------|---------|
| Train to Train | Podium ID & Development（潛力識別） | 全國錦標賽 |
| Train to Compete | Podium Potential（獎牌台潛力） | 世界盃 |
| Train to Win | Podium Success（獎牌台成功） | 世界錦標賽 / 奧運帕運 |

路徑構成要素：
- **Gold Medal Profile**：頂尖選手技能與能力數據
- **Podium Results Track**：通往獎牌台的表現路線
- **Winning Style of Play**：競賽策略與特質
- **Stage-specific Skills & Attributes**：ADM 詳述的具體發展指標

---

## 1.4 高水準表現（HP）的定義

### 奧運項目
**「全球前 8 名，並持續朝奧運獎牌台表現邁進。」**

### 帕運項目
- 全職投入，在關鍵賽事達成個人生涯最佳成績
- **全球前 5 名**，且持續進步

### 高水準表現三個層級（優先順序）
1. 奧運或帕運的獎牌台表現
2. FINA 世界錦標賽或世界帕拉游泳錦標賽獎牌台
3. 奧運 / FINA 世界錦標賽決賽（前 8）

### 高水準潛力識別標準（符合任一即可）
- 達到公布的「**On Track Times（達標時間）**」
- 在 FINA 世界青年錦標賽或青年泛太平洋錦標賽奪牌
- FINA 世界青年錦標賽前 8
- WPS 世界系列賽前 5

### 訓練季要求
- **以長池為主的訓練季**
- 一年通常**只有兩次巔峰**：春季選拔賽 + 夏季重點國際賽
- 遵循 **48 週以上年度訓練計畫（YTP）**

---

## 1.5 終身競技（Competitive for Life）

> 「從 U-14 社區聯賽到世界大師賽皆包含在內。致力於進步與勝利，並以此調整訓練方式。」

終身競技游泳員的特點：
- 並非定義上的 Podium Pathway 選手
- 常與 Podium Pathway 選手同場訓練
- 在省級與全國各類賽事中追求獎牌
- 包括大學游泳（U SPORTS）與加拿大游泳錦標賽的參賽者

---

## 1.6 發展的核心原則

### 發展不是線性的
每位運動員的發展速度與節奏都不同，影響體能、技術、心理與生活技能的發展。有經驗的教練能理解差異，為每位運動員創造適合的挑戰。

### 六種典型路徑

| 路徑 | 說明 |
|------|------|
| A | 從入門到世界錦標賽，始終專注同一項運動 |
| B | 較晚開始，青少年時期找到游泳，最終進軍奧運 |
| C | 早期在另一項運動取得成功，最終回歸游泳 |
| D | 在一項運動達到高水準後轉項，需重新從較低階段發展 |
| E | 參與多種運動，從未進入 Train to Train，但有優質終身活躍經驗 |
| F | 較晚才接觸運動，透過正向初次體驗走向終身活躍 |

# 第二章：ADM 四大支柱

## 2.1 什麼是 ADM？

**ADM** 是游泳員發展的廣義路線圖，描述在每個發展階段中，我們希望游泳員展現的觀察、行動與表達方式。

### ADM 不是什麼
- 不是建立訓練計畫的詳細處方
- 不是教練的「操作手冊」
- 不是評估運動員發展的測試組合

### ADM 的真正功能
- 為教練提供引導，協助創造適合發展的環境與挑戰
- 激發討論，思考如何判斷運動員進步及如何最佳支持其旅程

---

## 2.2 四大支柱

Swimming Canada 的 ADM 由四個相互關聯的支柱組成：

| 支柱 | 說明 |
|------|------|
| **體能（Physical Capacities）** | 在最能回應訓練刺激的時機，依發展階段進行適當訓練 |
| **技術／戰術／策略（Technical / Tactical / Strategic）** | 從基本動作到最高水準競賽的完整技能序列 |
| **心理技能（Mental / Psychological Skills）** | 依發展階段培養適合的運動心理技能 |
| **生活技能（Life Skills）** | 培養成為自信、自立個體所需的技能 |

> 注意：心理技能與生活技能章節是從**游泳員的視角**撰寫。

---

## 2.3 各支柱定義

### 體能（Physical Capacities）—「4S」

| 要素 | 說明 |
|------|------|
| **Stamina（耐力）** | 有氧與無氧持續能力 |
| **Strength（肌力）** | 最大肌力、肌耐力、爆發力、核心力量 |
| **Speed（速度）** | 局部速度、全身速度、無氧爆發力 |
| **Suppleness（柔軟度）** | 關節活動範圍、動態與靜態柔軟度 |

### 技術戰術策略

學習從基本人體動作到最高水準競賽所需的完整技能序列：基本人體動作 → 基礎動作技能 → 運動基礎技能 → 運動專項技能 → 戰術 → 策略

### 心理技能—六大核心

| 技能 | 說明 |
|------|------|
| **正向態度** | 預期正向結果，從挫折中學習 |
| **動機** | 持續改進的渴望與決心 |
| **目標設定** | 明確、可追蹤的短中長期目標 |
| **情緒識別與控制** | 辨識並調節情緒對表現的影響 |
| **專注力控制** | 在訓練與競賽中保持有效的注意力 |
| **減少錯誤** | 分析錯誤來源，發展改善策略 |

### 生活技能—SEL 五大核心能力

| 能力 | 說明 |
|------|------|
| **自我覺察** | 準確認識自身情緒、想法與優勢 |
| **自我管理** | 調節情緒、設定目標、面對挑戰 |
| **社會覺察** | 同理他人、欣賞多元 |
| **人際關係技能** | 溝通、合作、解決衝突 |
| **負責任的決策** | 依據道德標準做出建設性選擇 |

---

## 2.4 四大支柱的相互關係

**生活技能** 奠定基礎 → **心理技能** 得以強化 → **技術戰術** 得以執行 → **體能** 得到最佳支撐

生活技能的強健發展為心理技能提供土壤，進而使技術/戰術技能與體能訓練都能發揮最大效益。

# 第三章：審閱 ADM 的八大考量

## 考量一：運動員發展根植於人類發展

傑出的技術加上卓越體能仍然不夠。**創造力、彈性、自制力與紀律**——這些特質並非與生俱來，從出生後不久便開始發展，並持續至青春期與成年早期。

> 強健的**生活技能**發展，將為習得**心理技能**奠定基礎，並促進最佳的技術戰術與體能發展。

## 考量二：發展並非線性

每位運動員各有不同，其發展的速度與節奏高度個人化。具備充分知識的教練，能理解這種**個別差異性**，為每位運動員創造適合其發展階段的訓練體驗。

## 考量三：成長、發展與成熟（GDM）

了解運動員的 **GDM**（Growth, Development, and Maturation）有助於教練與家長提供符合發展階段的支持，促進安全、健康的人類發展。

## 考量四：整合支援團隊（IST）

從年輕游泳選手與其家庭開始建立。成員包括：教練、物理治療師、運動心理師、運動生理學師、運動營養師、醫療人員。

## 考量五：營養與水分補充

採取**積極補充能量習慣**的運動員，更可能：有效適應訓練刺激、獲得正向表現進步、維持強健骨骼健康、減少疾病或受傷頻率。

## 考量六：女性運動員的特殊考量

訓練環境若能做到以下三點，可觀察到更少傷害與更高留訓率：
1. **採用肌肉骨骼傷害預防計畫**
2. **推動積極的飲食補給習慣**
3. **培養接納與社群連結的氛圍**

## 考量七：有障礙的運動員

有障礙的運動員，首先也是最重要的身份是**運動員**。在幾乎各個層面，他們與非障礙同儕的共同點遠多於差異。

**先天性障礙**：經歷與一般運動員相同的發展階段，確保基本動作技能的發展最為關鍵。

**後天性障礙**：障礙發生高峰年齡為 15–19 歲與 20–29 歲，需要「重新學習 Active Start」，第一次接觸游泳俱樂部的體驗至關重要（無物理/溝通/課程/態度障礙）。

## 考量八：各階段表格的撰寫視角

- **體能、技術戰術**表格：**從教練的角度**撰寫
- **心理技能、生活技能**表格：**從游泳選手的角度**撰寫

### 執行功能（Executive Functions）

| 核心執行功能 | 說明 |
|------------|------|
| **工作記憶** | 在腦中保留並運用思維與資訊的能力 |
| **抑制控制** | 控制思維與衝動、抗拒分心與不良習慣的能力 |
| **認知彈性** | 切換思路、適應變化中情境與優先事項的能力 |

### 認知、情感與社會技能發展進程

| 發展階段 | 認知技能 | 情感技能 | 社會人際技能 |
|---------|---------|---------|------------|
| **幼兒期**（3–6 歲） | 基本計畫與目標設定 | 認識自己與他人的情緒 | 出現親社會行為（幫助、分享） |
| **兒童中期**（7–10 歲） | 較長期目標設定、組織資訊解決問題 | 理解並辨識情緒；出現同理心 | 理解社交線索、基本衝突解決 |
| **青春期前**（11–13 歲） | 計畫、組織、彈性思考更成熟 | 展現同理心與觀點採納 | 更複雜衝突解決；團隊合作 |
| **青春期**（14–17 歲） | 彈性應用認知技能於陌生任務 | 彈性應用情感技能於新情境 | 應對複雜社交情境；健康人際關係 |

貫穿所有發展階段的兩個核心維度：
- **品格與價值觀**：誠信、誠實、慈悲、勤奮、公民與倫理參與、責任感
- **態度、信念與心態**：自我概念、自我認識與身份認同；自我效能、動機與目的感
